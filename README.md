## ⚙️ Instalación del proyecto

1. Clonar repositorio:

```bash
git clone https://github.com/danigallard0/tamara_hoyas_beauty_studio.git
cd tamara_hoyas_beauty_studio
```

2. Instalar dependencias:

- composer install
- npm install

3. Crear archivo .env:

- cp .env.example .env

4. Generar clave de la aplicación:

- php artisan key:generate

5. Configurar la base de datos en el archivo .env

6. Ejecutar migraciones:

- php artisan migrate

7. Iniciar servidor:

- php artisan serve
